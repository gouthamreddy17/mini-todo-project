# Mini To-Do Project

A console-based To-Do List application developed in Python. This project helps users manage their daily tasks by providing features such as adding, updating, deleting, and viewing tasks. It also allows users to track completed and pending tasks through a simple menu-driven interface.

## Features
- Add Tasks
- Update Tasks
- Delete Tasks
- View All Tasks
- View Completed Tasks
- View Pending Tasks
- Duplicate ID Validation

## Technologies Used
- Python
- Dictionaries
- Functions
- Loops and Conditional Statements
